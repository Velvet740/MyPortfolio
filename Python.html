Python

Python is a high level and a general-purpose programming language. Its function includes code readability with significant indentation. It ranks as one of the most popular programming languages and used widely in machine learning.

Python’s statements include the following and more:
1. The “if” statement conditionally executes codes, along with else and elif (short form of else-if)
2. The “for” statement iterates over an iterable object, capturing elements to local variables
3. The “while” statement executes a block of code as long as the conditions are true
4. The “def” statement defines a function or method
5. The “assignment” statement which assigns values to a variable uses a single equals sign =

Python’s expressions include the following and more:
- Python uses expressions +, -, / , * and ** operators for mathematical operations - addition, subtraction, floating-point division, multiplication and exponentiation
- It uses “and”, “or”, and “not” as Boolean operators
- In Python, lists are written as [1, 2, 3]. They are mutable, and cannot be used as dictionary keys (they are immutable in Python). Tuples, written as (1, 2, 3), are immutable and can be used as dictionary keys if all the elements are immutable.
